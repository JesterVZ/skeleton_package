
Skeleton widget, which is used as a stub during data loading

## Features

This widget can be a good stub

## Getting started
 - install package
 - use in your code
```dart
loading || response == null ? const AwersomeSceleton( width: 100, height: 20, shape: SkeletonShape.rectangle)
```
